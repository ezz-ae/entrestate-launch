# **App Name**: EntreSite AI

## Core Features:

- AI Template Orchestration: Suggest optimal block sequences for high conversion, considering SEO and ads readiness. The system will use AI to analyze different block configurations to find the best tool for converting page viewers into leads.
- AI Content Design: Generate marketing copy for projects, including headlines, descriptions, and meta tags.
- AI-Powered Ad Generation: Generate Google Ads campaigns automatically from page content.
- AI Chat Assistant Builder: Build a chat assistant with vector embeddings generated from page and project data.
- SEO Automation: Auto-generate SEO metadata, OG, JSON-LD for real estate listings.
- Firebase Firestore Integration: Use Firestore as the primary database for storing project data, site configurations, and AI-generated content.
- Modular Site Building UI: Allow users to build sites using a drag-and-drop interface with AI-suggested blocks.

## Style Guidelines:

- Primary color: Deep teal (#008080) to reflect trust, stability, and sophistication in real estate. DO NOT USE TEAL. The real primary color should be Dark Blue (#002F4B) for a professional and trustworthy feel.
- Background color: Light grayish-blue (#F0F8FF), desaturated and bright, to create a clean and modern look.
- Accent color: Soft orange (#FFB347), analogous to yellow, used for calls to action and important highlights to add warmth and energy.
- Body and headline font: 'Inter' (sans-serif) for a modern and neutral aesthetic.
- Use clear, minimalist icons to represent various features and categories within the real estate listings.
- Employ a clean and structured layout with a focus on showcasing property images and key information.
- Use subtle transitions and animations to enhance user experience without being distracting.
